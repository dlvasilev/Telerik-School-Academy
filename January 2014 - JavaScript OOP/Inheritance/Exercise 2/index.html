Use The Console

School

To initalize school use: var schoolName = new School(name,town,classes); (classes must be array)

To get school's name use: schoolName.name;

To get school's town use: schoolName.town;

To get school's classes as array use: schoolName.classes;

To get school's classes as strign use: schoolName.classesStr;

To get school's full information as strign use: schoolName.schoolInfo();

Teacher

To initalize student use: var studentName = new Student(fname,lname,age,school var,grade)

To get student's introduction use: studentName.introduce();

To get student's first name use: studentName.fname;

To get student's last name use: studentName.lname

To get student's full name use: studentName.fullname

To get student's age use: studentName.age

To get student's grade use: studentName.grade

To get student's school name use: studentName.school.name;

To get student's school town use: studentName.school.town;
