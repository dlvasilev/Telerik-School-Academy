Use The Console

School

To create new school use: var schoolName = Object.create(School);

To initalize school use: schoolName.init(name,town,classes); (classes must be an array)

To get school's name use: schoolName.name;

To get school's town use: schoolName.town;

To get school's classes as array use: schoolName.classes;

To get school's classes as strign use: schoolName.classesStr();

To get school's full information as strign use: schoolName.schoolInfo();

Teacher

To create new school use: var teacherName = Object.create(Teacher);

To initalize teacher use: teacherName.init(fname,lname,age,school var)

To add teacher's speciality use: teacherName.spec(speciality)

To get teacher's introduction use: teacherName.introduce();

To get teacher's first name use: teacherName.fname;

To get teacher's last name use: teacherName.lname

To get teacher's full name use: teacherName.fullname()

To get teacher's age use: teacherName.age

To get teacher's speciality use: teacherName.spec

To get teacher's school name use: teacherName.school.name;

To get teacher's school town use: teacherName.school.town;
